# ReqRec

A Hypertext Transfer Protocol (HTTP) Request Recorder.
- Source code: {{pkg.repository.url}}

## A record is created for you:
record_id: {{record_id}}

# Usage

## Send a GET request to the record from browser:
{{self_url}}/records/{{record_id}}

## View the raw recorded requests in browser:
{{self_url}}/records/{{record_id}}/raw

## Send POST a request to the record from command line:
curl -d '{ "name": "Cicada" }' \
  -H "Content-Type: application/json" \
  "{{self_url}}/records/{{record_id}}"

# API

## Get the recorded requests from command line:
curl "{{self_url}}/api/records/{{record_id}}/requests"

## Delete the record from command line:
curl -X DELETE "{{self_url}}/api/records/{{record_id}}"
